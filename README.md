## 🚀 Overview

A **desktop-based text editor** with built-in **file compression and decompression using the Huffman Coding algorithm**.

This project combines:
- 🧠 Algorithms (Huffman Coding)
- 💾 File Handling
- 🖥️ GUI Development
- ⚙️ System Integration

---

## ✨ Features

### 📝 Text Editor
- Create, open, edit, and save `.txt` files  
- Cut, copy, paste, and select all  
- Simple and responsive interface  

### 🗜️ Compression
- Compress files using Huffman Coding  
- Generate `.huff` files  
- Efficient data encoding  

### 🔓 Decompression
- Restore original files from `.huff`  
- Accurate reconstruction of data  
- One-click decompression
